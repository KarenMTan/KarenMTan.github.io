1. Introduction to HTML and CSS

In order to write code, you will need to use a text editor. Text editors like Notepad on Windows or TextEdit on Mac will work. But using text editors Visual Studio Code or Sublime Text will not only allow you to edit your code, but also detect syntax errors, provide suggestions, and autofill properties.

HTML stands for “Hypertext Markup Language.” It’s used in conjunction with CSS and Javascript to create web pages and web applications.

CSS stands for “Cascading Style Sheet.” While HTML outlines the skeleton of your webpage, CSS focuses on the display and style. You can use CSS to simultaneously control the layout of multiple pages.

This first project will demonstrate basic HTML and CSS features.

  1. Open text editor
  2. Create a new file
  3. Save as “index.html”
  4. Insert the <!DOCTYPE html> declaration
    • The declaration tells your web browser that it should render an HTML document. It is always the first line of your HTML code.
  5. Type the <html></html> tags
    • The “<>” and “</>” represent opening and closing tags respectively. Not all HTML elements have a closing tag, but most do. If you’ve downloaded a development text editor, it may even auto-complete the closing tag for you!
  6. Type the <head></head> tags within the <html></html> tags
    • Information placed within the <head></head> tags is primarily metadata that will not be displayed to the user. For example, the font type or the external CSS style sheets are referenced within the <head></head> tags.
  7. Type the <body></body> tags within the <html></html> tags
    • Following most heads is a body - HTML is no exception. The <body></body> tags store the web page’s main content.
