1. Introduction to HTML and CSS

In order to write code, you will need to use a text editor. Text editors like Notepad on Windows or TextEdit on Mac will work. But using integrated development environments (IDEs) like Visual Studio Code will not only allow you to write code, but also detect syntax errors, debug code, and autofill properties.

HTML stands for “Hypertext Markup Language.” It’s often used in conjunction with CSS and Javascript to create web pages and web applications.

CSS stands for “Cascading Style Sheet.” While HTML outlines the skeleton of your webpage, CSS focuses on the display and style (as per the name). You can use CSS to simultaneously control the layout of multiple pages.

This first project will demonstrate basic HTML and CSS features.

  1. Open your IDE or text editor
  2. Go to "File" and select "New file"
  3. Save as “index.html”
  4. Insert the <!DOCTYPE html> declaration
    • The declaration specifies which version of HTML (HTML5) your document is in to your web browser. It is always the first line of your HTML code.
  5. Type the <html></html> tags. All your HTML code should go between these tags.
    • The “<>” and “</>” represent opening and closing tags respectively. Not all HTML elements have a closing tag, but most do.
    • If you downloaded VSCode, type the element followed by "Tab", this will add “<>” around the element followed by the corresponding closing tag.
  6. Type the <head></head> tags within the <html></html> tags
    • Information placed within the <head></head> tags is primarily metadata that will not be displayed to the user. For example, the font type or the external CSS style sheets are referenced in the <head></head> tags.
  7. Type the <body></body> tags within the <html></html> tags
    • The webpage's main content goes between the <body></body> tags.

    8. You can view your website by navigating to the folder you saved "index.html" in, then right-click and select "Open with" and choose your preferred browser. It's going to look empty right now, so let's add some content to your portfolio!

  8. Type <h1>Welcome!</h1> within the <body></body> tags
    • Headings are defined with <h1> through <h6> (similar to the styling you'd use in Microsoft Word).

    9. I encourage you to refresh your browser and see the changes to your website after every step!

  9. Type <p>My name is Your Name. I am excited to code my own website portfolio!</p> beneath your heading
    • These <p> tags represent paragraphs.
  10. Between the two sentences, type <br> (Within the <p> tag)
    • This <br> tag represents a line break. They can also be used between tags (in addition to within tags).
    • It is known as an empty tag because it has no closing tag.
  11. Type <p>Find more on my <a href="link/to_your/linkedIn">LinkedIn</a></p>
    • These <a> tags represent hyperlinks.
    • The "href" within the opening tag is stands for hypertext reference. It is an attribute that specifies the hyperlink destination. Attributes specify additional information about an element and often come in key-value pairs.
  12. Before the LinkedIn line, type <hr>
    • This <br> tag represents a thematic break. It will add a line between your name and the link to your LinkedIn.
    • This tag should be used to indicate a shift in topic.
  13. Before the thematic break, type <img src="file/destination-to-a/photo" alt="Description of photo" id="unique-name-here">
    • This <img> tag represents an image.
    • The src attribute marks the location of the image on your computer or on the internet. To get the link of an image on the internet, right-click and copy image address.
    • The alt attribute denotes alternative text when the image cannot be rendered. It also specifies what text should be read to users who use screen readers. To see the alternate text you entered, change the src to an invalid location.
    • The id attribute defines a unique ID to this element. No other element in your HTML file should have a matching id. It is not exclusive to images; any HTML element can have an id. We will use the id to style your image.
    • On a similar note, your elements can be referenced with the class attribute. This attribute is similar to id, but multiple elements in the file can be part of the same class. This will allow you to style multiple elements simultaneously.

    14. Notice the difference between the styling of the heading and paragraph. Although the text between the <h#> tags is larger and "bold" compared to the text between the <p> tags, you should not use the <h#> tags to make text larger and bold.

    Housing a heading within a heading tag and a paragraph within a paragraph tag practices semantic HTML. Praticing semantic markup is important for several reasons:

    Search engines use headings to catalouge the content of your website. Using headings to enlarge and emphasis text arbitrarily may prevent search engines from depicting your website accurately.

    Visually impaired users use screen readers to read the webpage to them. When a developer follows the best practices for web acessibility, more users can navigate your site as you intended.

    Now that you know some basics, let's learn CSS!

  14. Go to "File" and select "New file"
  15. Save as “style.css”
  16. Within the head tags, type <link rel="stylesheet" src="style.css">
  17. Let's first change the font of the webpage:
    • Go to Google Fonts and select a font for your website.
    • Alternatively, CSS comes with web safe font combinations. You can find the list on w3schools.com.
  18. Now that you've selected a font, go to style.css and type:
    • If you chose a font from Google Fonts, copy the code in the "Specify in CSS" section. Additionally, in the head tag of your HTML file, just as you linked the external CSS, link the font by copying the code in the "Embed Font" section.
    • If you chose a font from the list of web safe fonts, copy the font-family and paste it in your style.css file.
    • Notice that only paragraph text changed. Repeat steps 17 and 18 to change your heading font.
  19. To adjust the size of your text, use the font-size property.
    • Refer to CSS units on w3schools.com.
  20. To adjust the size of your image, type a "#" followed by the id of your image followed by "{ }". Within the braces use the height and weight properties.
    • To maintain the image's original ratio, explicitly specify height and set width: auto (and vice versa).
    • You don't need to refer to the image with an id. Instead you can use the element name "img" as a selector as well. However, this may get tricky if your webpage has multiple images that you want to size differently.
    • You can find more image styling techniques on w3schools.com.
